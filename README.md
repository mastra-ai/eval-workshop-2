# Mastra Evals Workshop

A hands-on workshop for learning how to evaluate AI agents using [Mastra](https://mastra.ai/).

## Demos

### demo-1-basics

A basic Mastra project demonstrating foundational concepts: agents, tools, workflows, and scorers. Good for getting familiar with the Mastra framework.

### demo-2-docs-agent

The main workshop content. Features a documentation agent with a complete evaluation pipeline including:

- Custom scorers for detecting failure modes
- Dataset creation and experiment running
- Online and offline evaluation strategies
- CI integration for automated testing

See [demo-2-docs-agent/README.md](./demo-2-docs-agent/README.md) for setup instructions and workshop guides.
